# Learn Pandas in One Day

## 🚀 Learning Roadmap

To master the material effectively, it is recommended to run the files in the following order:

1. **`importing.py`** — Installing Pandas and your first library import.
2. **`series.py`** — Creating one-dimensional arrays (Series) and working with indexes.
3. **`dataframes.py`** — Dataframe basics: creating tables, adding rows, and columns.
4. **`selection.py`** — Navigating data: using `.loc` (label-based) and `.iloc` (index-based).
5. **`filtering.py`** — Selecting data based on conditions (filtering).
6. **`aggregation.py`** — Data analysis: means, sums, and grouping (`groupby`).
7. **`data_cleaning.py`** — Data cleaning: handling missing values (NaN), removing duplicates, and formatting.

## 📂 Project Structure

* **`src/`** — Source code for all lessons.
* **`data.csv`** — Sample dataset (Pokemon stats) used in the lessons.
* **`.venv/`** — Python virtual environment.
